# Algorithm Design Lab 🧠💻

This repository contains my **Algorithm Design laboratory assignments** implemented in **Python**.  
The programs demonstrate fundamental concepts of **data structures, algorithms, and optimization techniques** commonly studied in algorithm design courses.

---

## 📚 Topics Covered

This repository includes implementations of various algorithms such as:

- Sorting Algorithms
  - Merge Sort
  - Quick Sort
  - Counting Sort
  - Radix Sort
  - Bucket Sort

- Graph Algorithms
  - Dijkstra's Shortest Path
  - Kruskal's Minimum Spanning Tree
  - Graph Coloring using Backtracking
  - Hamiltonian Cycle

- Greedy Algorithms
  - Fractional Knapsack

- Dynamic Programming
  - 0/1 Knapsack Problem
  - Binomial Coefficient (Recursive, Top-Down DP, Bottom-Up DP)

- Data Structures
  - Binary Search Tree Operations (Insert, Delete, Traversals)
  - Merge Two Sorted Linked Lists

- Searching Algorithms
  - Binary Search Variations
  - Search in Rotated Sorted Array

- Other Problems
  - Palindrome Number
  - Element occurring three times in array
